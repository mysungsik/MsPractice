[미디어쿼리란?] 화면의 크기가 일정수준 이하부터 이상일 경우 실행되는 [반응형 크기조절 코드] 1. desktop first 2. mobile first 둘의 차이점 = desktop first는 max-width 지정하여, 작아질때 미디어쿼리 시작 : 데스크탑은 애초에 크게보고, 작아지면 줄어드는 것이기때문 mobile first는 min-width 지정하여, 커질때 미디어쿼리 시작 : 모바일은 애초에 작게보고, 크기를 키울때 변해야하기 때문 * max-width : 미디어쿼리의 한계점이 거기까지 [나 1000px까지 밖에 못써!] * min-width : 미디어쿼리의 한계점이 거기서부터 시작 [나 1000px 부터 시작하는 미디어쿼리야!] [미디어쿼리가 시작되면] : 기존의 코드는 그대로 유지되대, 미디어쿼리에 적은 코드가 있다면 그것만 변형됌 ex) 기존 코드 쫙, 미디어쿼리가 시작되면 p의 폰트크기를 바꿔라 라고 지정했다면 화면이 작아질경우 미디어쿼리가 시작되어 기존의 코드에서 p의 크기만 변한다.
